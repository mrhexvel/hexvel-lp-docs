# Основы использования

## Символика

- **±**: Команда работает с "+" (включить) или "−" (выключить).
- Пример: `±лайк` → `+лайк` (поставить лайк) или `-лайк` (убрать).

## Аргументы

- **()**: Обязательный параметр.
- **""**: Необязательный, с дефолтным значением.

## Переменные

| Переменная     | Описание                                                | Пример       |
| -------------- | ------------------------------------------------------- | ------------ |
| `ссылка`       | Линк или тег пользователя                               | `vk.com/id1` |
| `текст`        | Текст для команды                                       | `Привет`     |
| `кол-во`       | Число                                                   | `5`          |
| `фото`         | Прикреплённое изображение                               | -            |
| `город`        | указать город                                           | -            |
| `интервал`     | указать время, после которого команда будет срабатывать | -            |
| `аудио `       | прикрепить аудиофайл                                    | -            |
| `префикс`      | указать префикс, который установлен в модуле            | -            |
| `тип префикса` | указать тип префикса для его изменения                  | -            |
| `here/all`     | `here` - в одной беседе, `all` - во всех                | -            |
| `hex код`      | это обозначения цветов для виртуального интерфейса      | -            |
